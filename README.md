# Secure Task Manager

A Python-based task management system for small businesses.

## Features

- Add, assign, and track tasks for team members
- User registration and login system
- Admin dashboard for user management
- Task status tracking (Pending, In Progress, Completed)

## Tech Stack

- Python 3.x

## Installation

```bash
git clone https://github.com/ZeakMeadows/secure-task-manager.git
cd secure-task-manager
python task_manager.py
```
